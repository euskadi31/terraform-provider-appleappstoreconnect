# appleappstoreconnect_subscription_group_localization (Resource)

Manages a localized name for a subscription group in App Store Connect.

## Example Usage

```hcl
resource "appleappstoreconnect_subscription_group_localization" "en" {
  subscription_group_id = appleappstoreconnect_subscription_group.premium.id
  locale                = "en-US"
  name                  = "Premium Club"
}
```

## Schema

### Required

- `locale` (String) The locale of the localization (e.g., `en-US`, `fr-FR`). Changing this forces a new resource.
- `name` (String) The localized name of the subscription group.
- `subscription_group_id` (String) The ID of the subscription group this localization belongs to. Changing this forces a new resource.

### Optional

- `custom_app_name` (String) An optional custom app name displayed for this subscription group in this locale.

### Read-Only

- `id` (String) The unique identifier of the subscription group localization.
