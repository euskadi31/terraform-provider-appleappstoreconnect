# appleappstoreconnect_subscription_localization (Resource)

Manages a localized name and description for an auto-renewable subscription in App Store Connect.

## Example Usage

```hcl
resource "appleappstoreconnect_subscription_localization" "en" {
  subscription_id = appleappstoreconnect_subscription.monthly.id
  locale          = "en-US"
  name            = "Premium Monthly"
  description     = "Unlimited access to all premium features."
}
```
