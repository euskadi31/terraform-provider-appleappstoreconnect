# appleappstoreconnect_app (Data Source)

Use this data source to retrieve information about an app in App Store Connect, either by its App Store Connect ID or by its bundle ID. This is the recommended way to obtain the app ID required by the In-App Purchase and subscription resources.

## Example Usage

### Find by Bundle ID

```hcl
data "appleappstoreconnect_app" "example" {
  bundle_id = "com.example.app"
}

output "app_id" {
  value = data.appleappstoreconnect_app.example.id
}
```

### Find by ID

```hcl
data "appleappstoreconnect_app" "example" {
  id = "1234567890"
}
```

### Use as the source for an In-App Purchase

```hcl
data "appleappstoreconnect_app" "prod" {
  bundle_id = "com.example.app"
}

resource "appleappstoreconnect_in_app_purchase" "premium" {
  app_id               = data.appleappstoreconnect_app.prod.id
  product_id           = "com.example.app.premium"
  name                 = "Premium"
  in_app_purchase_type = "NON_CONSUMABLE"
}
```

### Optional

- `bundle_id` (String) The bundle ID of the app (e.g., `com.example.app`). Provide either `id` or `bundle_id`.
