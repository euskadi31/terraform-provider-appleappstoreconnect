# appleappstoreconnect_subscription (Data Source)

Use this data source to retrieve information about an existing auto-renewable subscription, either by its ID or by subscription group ID and product ID.

## Example Usage

### Find by ID

```hcl
data "appleappstoreconnect_subscription" "example" {
  id = "1234567890"
}
```

### Find by group and product ID

```hcl
data "appleappstoreconnect_subscription" "example" {
  filter = {
    subscription_group_id = appleappstoreconnect_subscription_group.premium.id
    product_id            = "com.example.app.premium.monthly"
  }
}
```
