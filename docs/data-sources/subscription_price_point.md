# appleappstoreconnect_subscription_price_point (Data Source)

Resolves a subscription price point ID from a customer price and territory. The resulting `id` is referenced by the `appleappstoreconnect_subscription_price` resource.

## Example Usage

```hcl
data "appleappstoreconnect_subscription_price_point" "usd_499" {
  subscription_id = appleappstoreconnect_subscription.monthly.id
  territory       = "USA"
  customer_price  = "4.99"
}
```

## Schema

### Required

- `customer_price` (String) The customer price to match (e.g. `4.99`), as returned by the API for the territory.
- `subscription_id` (String) The ID of the subscription whose price points are searched. Price points are specific to each subscription.
- `territory` (String) The territory code to look up the price in (e.g. `USA`).

### Read-Only

- `id` (String) The resolved price point ID.
- `proceeds` (String) The proceeds (developer revenue) for the resolved price point.
